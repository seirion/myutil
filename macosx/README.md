### Homebrew 설치
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew update
```

### pip 설치
```
sudo easy_install pip
```

### ipython 설치
```
pip install ipython
```

### 화면 캡쳐
- 전체 : shift + command + 3
- 영역 : shift + command + 4
- 창 : shift + command + 4 + space
